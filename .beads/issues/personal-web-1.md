## Description

Portfolio web berbasis Next.js dan React yang dirancang sebagai representasi profesional digital untuk menampilkan portofolio, pengalaman profesional, dan personal branding secara konsisten, modern, dan scalable.

### Goals:
- Meningkatkan profesionalitas personal brand
- Mencari opportunity job & career growth (online resume/CV interaktif)
- Fondasi untuk pengembangan konten (blog/news section)

### Target Audience:
- Client (freelance/agency/corporate)
- Recruiter & HR
- Hiring Manager / Tech Lead
- Professional collaborator

### Tech Stack:
- Next.js 14, React 18, TypeScript 5
- Tailwind CSS 3, Framer Motion
- React Icons, React Hook Form + Zod

## Acceptance Criteria

- [ ] Next.js project structure setup dengan semua dependencies
- [ ] Global design system (colors, typography, spacing) identik di semua page
- [ ] Core layout components (Sidebar, Header, Layout)
- [ ] Home page dengan hero section, circular profile image, profession cycling
- [ ] About page dengan bio, personal info grid, CV download
- [ ] Service page dengan 6 service cards grid
- [ ] Responsive design untuk mobile dan desktop
- [ ] Performance optimization (image optimization, code splitting)
- [ ] SEO optimization dan accessibility compliance
- [ ] Smooth animations dan interactions
