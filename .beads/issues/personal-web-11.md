## Description

Implement SEO optimization dan accessibility features untuk better search visibility dan user experience.

### SEO Tasks:

**1. Meta Tags:**
- Title tags (unique per page)
- Meta descriptions
- Open Graph tags (for social media)
- Twitter Card tags
- Canonical URLs
- robots.txt
- sitemap.xml

**2. Structured Data:**
- JSON-LD schema markup
- Person schema
- Organization schema
- WebSite schema

**3. SEO Best Practices:**
- Semantic HTML structure
- Proper heading hierarchy (H1, H2, H3)
- Descriptive URLs
- Internal linking
- Image alt texts
- Fast loading speed

### Accessibility Tasks:

**1. Semantic HTML:**
- Use proper HTML5 elements: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`
- Proper heading hierarchy
- Landmark roles

**2. ARIA Labels:**
- ARIA labels untuk interactive elements
- ARIA roles untuk custom components
- Screen reader announcements

**3. Keyboard Navigation:**
- Clear focus states
- Tab order
- Skip navigation link
- Keyboard shortcuts (optional)

**4. Visual Accessibility:**
- Color contrast: Min WCAG AA standard (4.5:1 untuk text)
- Focus indicators
- Alt text untuk semua images
- Readable font sizes
- No reliance on color alone

**5. Testing:**
- Test dengan screen readers
- Keyboard-only navigation test
- Lighthouse accessibility audit
- WAVE accessibility tool
